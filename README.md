# DoNote

- Note it. Do it. Win donut 
Do-Note helps you get things done. 

LIVE: [do-note-amiri](https://github.com/do-note-amiri.vercel.app)

Features: 
- Damn easy to use! (I really tried hard on the UX)
- Minimalist design
- Clear flow to help you just focus on your TASKS and get things DONE.
- Filter Done tasks, and Doing tasks or see ALL by default
- Managing multiple accounts 
- Auto theme toggling based on your browser/os setting

NOTES: 
- App is on phase 1(testing)
so it does not have data peresistance(refreshing wipes out the data) but in phase 2 we are adding data peresistance to make it complete and really usable.
